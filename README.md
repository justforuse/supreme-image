# Supreme Image

> 制作Supreme风格的图片

## Features
- 支持部分图片渲染功能：模糊和灰度
- 内置三种风格的supreme图片
- 可调整supreme的垂直方向的位置（目前居中显示），大小，透明度
- 保存图片，使用[html2canvas库](https://github.com/niklasvh/html2canvas)保存图片，但是无法保存图片的灰度和模糊，需要手动截图。。
